# Whisker Dash

## Overview

**Whisker Dash** is a personal game project developed using **Godot 4.2.2**. This game is a charming platformer where players control a playful kitty navigating through beautifully crafted levels. Drawing inspiration from the acclaimed game **Celeste**, **Whisker Dash** combines challenging gameplay with heartwarming visuals and engaging mechanics.

**Note:** This project is currently in progress and many more features are planned for future implementation!

## Features

- **Challenging Levels**: Navigate through intricately designed levels that test your platforming skills.
- **Level Switching System**: Seamlessly transition between different levels, enhancing the gameplay experience and providing varied challenges.
- **Intuitive Controls**: Simple yet responsive controls that make jumping, climbing, and dashing a delight.
- **Beautiful Art Style**: Vibrant graphics and animations that bring the world of Whisker Dash to life.
- **Engaging Soundtrack**: A captivating musical score that complements the game's atmosphere and keeps players immersed.

## Installation

1. **Download Godot**: Make sure you have **Godot 4.2.2** installed on your machine.
2. **Clone the Repository**: Use Git to clone the repository to your local machine.
3. **Open in Godot**: Launch Godot and open the project by selecting the directory where you cloned the repository.
4. **Run the Game**: Press the Play button in Godot to start your adventure with Whisker Dash!
